Project Details

Clothing Collection E-commerce Website

About This Project

This was my first complete e-commerce website built to practice HTML, CSS, and JavaScript. The goal was to create a functional clothing store that works well on both desktop and mobile devices.

Features Implemented

  • Responsive design that works on all screen sizes
  • Product catalog with categories (Men, Women, Kids)
  • Shopping cart functionality with add/remove items
  • Product filtering by category and price
  • Clean, modern UI with hover effects
  • Contact form for customer inquiries

What I Learned

  • How to structure an e-commerce website
  • Implementing responsive design with CSS Grid and Flexbox
  • Basic JavaScript for interactive features
  • Creating a shopping cart with local storage
  • Deploying a website to Netlify

Technologies Used

HTML5 CSS3 JavaScript (ES6) CSS Grid Flexbox Local Storage API Netlify
